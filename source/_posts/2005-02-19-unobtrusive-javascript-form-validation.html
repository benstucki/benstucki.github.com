I've been reading allot about 'unobtrusive javascript' lately. Stuart Langridge has a popular table sorting script using unobtrusive javascript, and John Dowdell from Macromedia even gave it a brief mention recently. It looks like it's been around for a while, but I guess it has gotten renewed interest as the older browsers dissappear. Just like seperation of style and content, it still may not always be applicable for business use yet, but any browser from the past few years should support the js functions needed. Anyways, I thought I should write an unobtrusive javascript file myself, and since I'm also a strong believer in client side form validation, I bring you validation.js. It provides a few formatting and validation functions for common form inputs such as phone numbers, credit cards, zip codes, etc. I'm honestly at a loss to understand why this type of client side formatting isn't used more often, but now that it's so easy to impliment there should be no excuse. Simply include the .js file and assign the related class to the appropriate input. You can view a demo here.
