Another week has passed and more entirely free, open source Flex 2 components are ready for download. This week I spent some time porting my CSSManager and CSS3Box AS2 components into AS3 for Flex. These components are considerably more involved than in previous weeks and hopefully more useful as well.

The CSSManager Component handles run-time CSS parsing in Flex. It includes methods for parsing CSS text and files as well as defining custom keyword values and shorthand properties. It cooperates with the existing StyleManager framework in Flex and allows component developers to take advantage of its ability while not requiring its use. I can't overstate the impact I think this could have for Flex 2. At the moment, however, it's not case sensitive and can't handle Flex specific or custom objects, only standard CSS values.

The CSS3Box Skin is a border skin that implements a small subset of the CSS3 Backgrounds and Borders Module working draft. It provides the ability to set border color, width, and corner radius values for each border and corner independently. I didn't get around to updating any child components of their new boundaries, but I really just needed it to demonstrate how the CSSManager class is used.
