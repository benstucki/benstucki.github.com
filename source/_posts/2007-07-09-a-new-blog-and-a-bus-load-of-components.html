As the Adobe onAIR Bus Tour rolls from town to town, I'll be playing along at home by posting a bus-load of components and code samples. Each time the bus stops for a full day event, I'll release a new component or library. Each night event is good for a code sample or technical post. This means I'll be releasing eight components and ten code samples over the next three months.

I've updated my blog to get ready for all the action and managed to hassle Chafic into getting the new Atellis Labs up and running last week. I even conned my way onto the FlexLib contributors list a while back. You can expect a lot of the components to find a rightful home outside of this blog, although I plan to talk about each of them here.

Just like the bus tour, the component releases will kick off tomorrow, keep trucking through August (including a big release at 360Flex) and lead up to Adobe Max in September.

Updated 05/28/2008 So I didn't get most of these blog posts out the way I wanted to, but you can see a lot of the code finding it's way into the community through projects such as Degrafa, OpenFlux, and Metaphile. There's also a number of unfinished components and experimentation available through my personal googlecode project.
