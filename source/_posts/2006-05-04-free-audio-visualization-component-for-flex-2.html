This week's entirely free and open source download is an easy to use audio visualization component for Flex 2 Beta 2. The visualization component can display sound data in multiple formats and includes custom events and properties for using advanced bitmap effects without changing the component class itself. It draws dynamically based on size and keeps memory consumption to a minimum.
