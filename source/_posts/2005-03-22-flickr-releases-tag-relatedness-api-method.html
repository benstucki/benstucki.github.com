--- type: post published: true status: publish tags: - General - Archive title: Flickr Releases Tag Relatedness API Method layout: post --- Flickr Releases Tag Relatedness API Method [ Tue, 22 Mar 2005 03:53:08 AM GMT ]
Ludicorp has been getting allot of attention due to their recent acquisition by Yahoo, and it hasn't all been good. So perhaps it is just good timing that a number of Flickr API wishes have been granted recently, including the release of a tag relatedness api method. Apparently the relatedness algorithm considers countless user interactions to learn how tags are related. Which makes this the first api method to reveal Flickr's social intelligence to independent developers. Whereas any other method is fairly strait forward in getting an individual's photos or contacts, this one represents the human equation which has created so much buzz. I think it shows Flickr's willingness to share a little bit of it's genius with the world and gives me assurance that I will be Flickring for quite a while longer. Now if only we could play with that interestingness api.
