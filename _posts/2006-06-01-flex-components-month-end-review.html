I've been blogging about Flex Components (mostly just posting examples) all month long, and according to my new blog format it's time to wrap things up and move on to next month's subject. This month was code heavy with examples posted for Audio Visualizations, Visual Reflections, 3D Bitmaps, Runtime CSS with CSS 3 Borders, AutoScroll Galleries and Metadata Parsing in Flex 2. Although most of the components used in these examples aren't complete, I feel relieved that they are finally out in the community, and I hope that they are improved upon and used regularly. I have posted the full com.fusiox source library to make it easier to download and explore. Here is how it breaks out.

com.fusiox.media.ID3Reader
The ID3Reader Component is used to parse the raw bytes of an ID3 tag into an array of AS3 objects. It is capable of parsing frame types that the core ID3 object can't yet, including embedded images.
com.fusiox.media.ID3Stream
The ID3Stream Component is used to download the ID3 portion of an MP3 file and load it into an ID3Reader instance. This allows users to view the MP3's metadata quickly without having to download the whole file.
com.fusiox.skins.CSS3Box
The CSS3Box Skin is a border skin that implements a small subset of the CSS3 Backgrounds and Borders Module working draft. It provides the ability to set border color, width, and corner radius values for each border and corner independently.
com.fusiox.styles.CSSManager
The CSSManager Component handles run-time CSS parsing in Flex. It includes methods for parsing CSS text and files as well as defining custom keyword values and shorthand properties. It cooperates with the existing StyleManager framework in Flex and allows component developers to take advantage of its ability while not requiring its use.
com.fusiox.ui.Bitmap3D
The Bitmap3D component is used to display any BitmapData object in 3D. Methods and properties are available for changing the rotation angles, image quality (for speed) and pixel by pixel depth values.
com.fusiox.ui.Image
The Image component extends the core Image class to load an image from a ByteArray.
com.fusiox.ui.Pixel3D
The Pixel3D component extends Bitmap3D and manipulates each pixel's depth based on color.
com.fusiox.ui.Reflection
The reflection component is used to give the illusion of a reflective surface in your Flex 2 applications. It includes a few properties for custom fading, skewing and scaling and was made with real-time reflection in mind.
com.fusiox.ui.Scroller
The Scroll component will render a horizontal display container that scrolls according to your mouse position. You should provide it with a dataProvider and itemRenderer and it will do the rest.
com.fusiox.ui.Visualization
The visualization component can display sound data in multiple formats and includes custom events and properties for using advanced bitmap effects without changing the component class itself. It draws dynamically based on size and keeps memory consumption to a minimum.
