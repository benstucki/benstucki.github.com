I've been updating some of my older work for the Flex 2 Beta 3 Player and wanted to point out that my Pixel3D Experiment is now updated for Beta 3 and free for any use. It includes two components, Bitmap3D and Pixel3D. The Bitmap3D component is used to display any BitmapData object in 3D. Methods and properties are available for changing the rotation angles, image quality (for speed) and pixel by pixel depth values. The less useful Pixel3D component extends Bitmap3D and manipulates each pixel's depth based on color. Any help that can be given to increase the speed or capability of these components is much appreciated.
